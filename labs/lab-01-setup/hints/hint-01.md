# Hint 01 — Lab 01

Un bon `CLAUDE.md` tient sur une page. Pose-toi 3 questions :

1. **Que fait ce projet ?** (1-2 lignes)
2. **Quelles conventions doivent être respectées ?** (ex : ESM, tests avec `node:test`, pas de dépendance externe)
3. **Quelles commandes reviennent souvent ?** (dev, test, lint, build)

Ne recopie pas le README. Ce que tu écris dans `CLAUDE.md` est **chargé à chaque session** : privilégie les contraintes, pas les descriptions.
