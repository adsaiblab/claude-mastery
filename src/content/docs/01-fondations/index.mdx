> Durée estimée : **2h** · 5 modules + 1 lab

Tu vas apprendre **comment Claude Code pense** avant d'apprendre comment l'utiliser. C'est le socle qui rend tout le reste intuitif.

## Modules

1. **Les modèles** — Opus 4.7, Sonnet 4.6, Haiku 4.5 : quand utiliser quoi.
2. **L'agentic loop** — les 3 phases qui transforment un LLM en agent.
3. **Le context window** — ta ressource principale, gérée activement.
4. **CLAUDE.md** — les instructions que tu écris, hiérarchie de chargement.
5. **Mémoire & sessions** — auto-memory, scopes, fork, resume.

## Lab 01 — Setup complet

À la fin du niveau, tu configures ton premier projet *from scratch* : `CLAUDE.md`, `.claude/settings.json`, permission modes, première session productive. Feedback binaire via `validate.sh`.
