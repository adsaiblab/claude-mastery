## Le pattern de chaque module

Chaque module suit un format invariant : **Situation → Concept → Démo → Check → Quiz → Référence**.
Aucune phrase ne commence par *"Dans ce module, nous allons…"* — on t'expose un scénario concret, puis la solution.
