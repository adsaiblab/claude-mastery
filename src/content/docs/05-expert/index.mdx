> Durée estimée : **5h** · 4 modules + 1 lab · Section Expert

<Situation>
  Tu maîtrises les bases (slash, hooks, subagents, MCP, plugins, production). Ce
  niveau te donne les **techniques de collaboration avancée** observées chez les
  utilisateurs experts : ingénierie du contexte à grande échelle, cycles de
  planification rigoureux, co-design en pair, catalogues de patterns
  réutilisables. Pas de magie — de la discipline.
</Situation>

## Modules

1. [**Context Engineering**](/05-expert/context-engineering/) — sélection, compression, stratification du contexte. Tirer le maximum de la fenêtre.
2. [**UltraPlan / UltraReview**](/05-expert/ultraplan-ultrareview/) — cycle 4 phases (Plan → Review → Implement → Judge) avec critères de sortie explicites.
3. [**Co-work Claude Design**](/05-expert/cowork-claude-design/) — Claude comme pair designer : critères, ADR, diagrammes Mermaid, désaccord explicité.
4. [**Advanced Patterns**](/05-expert/advanced-patterns/) — catalogue : doer/judge, fan-out/fan-in, writer/reviewer, mémoire cross-session, cascade de modèles.

## Lab 05 — Mon système d'agents

Assembler ton propre setup multi-agent sur un projet réel : subagents typés, hooks, MEMORY, cycle UltraPlan/UltraReview → [Lab 05](/05-expert/lab-05/).

### Lab 05 alt — Capstone freeform

Variant capstone : sors du sandbox et pose Claude Code sur ton vrai repo. Tu choisis un archétype (audit, refactor, agent custom, routine), tu scopes, tu livres, tu débriefes → [Lab 05 alt](/05-expert/lab-05-capstone/).

## Après ce niveau

Tu sauras :

- Surveiller et dompter ton context window (5 couches, /compact, fork, mémoire typée).
- Mener un cycle UltraPlan/UltraReview avec budget tokens et critères de sortie.
- Co-designer une archi avec Claude — critères partagés, ADR, pre-mortem.
- Composer des patterns multi-agent (doer/judge, fan-out, writer/reviewer, cascade) selon le besoin.
- Capitaliser cross-session via une discipline de mémoire typée.
