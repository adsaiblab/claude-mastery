> Durée estimée : **4h** · 6 modules + 1 lab

Claude sort du terminal : CI, backends, pipelines non-interactifs, agents managés. Ce niveau te donne les outils pour **embarquer Claude en production** sans perdre le contrôle.

## Modules

1. **GitHub Actions** — exécuter Claude dans un workflow (issue triage, PR review).
2. **Agent SDK — Python** — lancer des conversations programmatiques côté backend.
3. **Agent SDK — TypeScript** — version Node, streaming, tool use.
4. **Sandboxing** — conteneurs, FS restrictions, network policies.
5. **Routines** — tâches planifiées, cron, triggers webhooks.
6. **Managed Agents** — Claude as a Service, scaling, quotas.

## Lab 04 — Claude in CI

Déployer un GitHub Action qui review chaque PR avec Claude, ajoute un commentaire structuré, et bloque le merge si score < seuil.
