## Concept — 10 catégories

Les flags de `claude` se rangent en 10 familles. Ce module donne les **plus utiles de chaque famille**. La [Référence CLI flags](/reference/cli-flags/) a la liste exhaustive.

## Combos typiques

**Audit lecture-seule sur un repo inconnu** :

```bash
claude --permission-mode plan --allowed-tools "Read,Grep,Glob,WebFetch" \
       --name audit-$(basename $PWD)
```

**CI headless pour PR review** :

```bash
echo "Review this PR diff: $(git diff origin/main)" \
  | claude -p --model sonnet --output-format json \
           --allowed-tools "Read,Grep,Bash(git:*)"
```

**Debug d'un hook qui déconne** :

```bash
claude --debug --name hook-debug
```

**Session de refacto sensible** :

```bash
claude --permission-mode plan --model opus --name refacto-auth-042026
# ... tu planifies, tu valides ...
# /mode acceptEdits     (une fois le plan OK, tu passes en accept-edits)
```
