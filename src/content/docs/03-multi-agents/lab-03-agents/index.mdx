<Situation>
  Variant du Lab 03. On reprend les subagents — mais cette fois pas en pipeline
  séquentiel : en **fan-out**. Tu as un monorepo avec 3 services indépendants à
  auditer. Plutôt qu'un explorateur en série qui mange ton context, tu dispatches
  un subagent **auditor** par service, en parallèle, puis tu agrèges. Le
  pattern qui permet d'auditer 50 modules sans saturer la fenêtre.
</Situation>

## Objectif

Livrer un setup où une slash-command `/audit-all` :

1. Détecte automatiquement les services dans `services/*`.
2. Lance **un subagent `auditor` par service en parallèle** (fan-out).
3. Chaque auditor produit `audit/<service>.md` avec score + concerns.
4. Un subagent `aggregator` fusionne les rapports en `audit-report.md` (table récap).
5. Un hook **PreToolUse** garde-fou bloque toute écriture hors `audit/` par les auditors.

## Démarrer

```bash
cd labs/lab-03-agents
./setup.sh              # crée work/ avec monorepo de démo + starter
cd work/
```

Tu dois créer dans `.claude/` :

- `agents/auditor.md` — read-only (Read, Grep, Glob), produit `audit/<service>.md`.
- `agents/aggregator.md` — Read + Write, produit uniquement `audit-report.md`.
- `commands/audit-all.md` — slash-command qui dispatche le fan-out.
- `hooks/guard-readonly.sh` — bloque `auditor` s'il tente Edit/Write hors `audit/`.
- `settings.json` — déclare le hook + permissions.

## Ce qu'on vérifie vraiment

1. `auditor.md` a `tools: Read, Grep, Glob` strictement (pas d'Edit/Write/Bash).
2. `aggregator.md` peut écrire mais pas exécuter Bash.
3. `commands/audit-all.md` mentionne les 3 services ET un pattern de parallélisation (plusieurs `Agent()` ou `@auditor` répétés).
4. `hooks/guard-readonly.sh` parse le payload PreToolUse et exit 2 si `tool_input.file_path` n'est pas dans `audit/`.
5. `settings.json` déclare `PreToolUse` avec matcher `Edit|Write` ET le hook.
6. La structure `audit/` existe et est versionnée (ou créée par setup).
7. Pas de fan-out faux : si la commande lance les auditors **en série**, le test échoue (heuristique : la commande utilise un wording de parallélisme).
